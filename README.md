Curelex Healthcare Website
Welcome to the Curelex healthcare website. This is a simple static website built using HTML, CSS, and JavaScript, designed to provide information and services related to healthcare.

Project Structure

index.html: The main homepage of the Curelex website.
css/: Contains stylesheets to make the website visually appealing.
js/: Contains JavaScript files that add interactivity and functionality.
pages/: Contains additional pages related to healthcare services, contact, and other information.

Features
A static, responsive website focused on healthcare-related information.
Simple and clean design with easy navigation.
Interactive elements powered by JavaScript.
Mobile-friendly layout with CSS for different screen sizes.

Usage

To view the Curelex healthcare website:
Download or clone the repository.
Open the index.html file in any modern web browser.

No installation is required.
