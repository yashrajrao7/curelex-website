Curelex – Smart Healthcare Web Application

📌 Overview
Curelex is a responsive healthcare web application built using HTML, CSS, and JavaScript. The platform includes home page along with a structured dashboard interface.

The project focuses on clean UI design, responsive layouts, and scalable frontend architecture while simulating a real-world healthcare system workflow.

---

🚀 Features

 🏠 Homepage
- Modern and responsive landing page
- Clean navigation structure
- Informative healthcare sections

📊 Dashboard
- Structured dashboard layout
- Role-based interface design
- Organized UI components for future scalability

---

 💻 How to Run the Project

1. Clone the repository:
   git clone https://github.com/SowmithPaida/curelex-website.git
  

3. Open the project folder.

4. Open `index.html` in any modern web browser.

No additional installation is required.
